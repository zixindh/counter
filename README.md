## RMB Counter Quick Guide

- **Goal**: Track each user's total RMB with quick add buttons and custom entries.
- **Run the app**:
  - Install Streamlit: `pip install streamlit`
  - Start the server from `workspace`: `streamlit run app.py`
- **Login**: Enter a name and press **Start counting**; the app remembers totals in `user_data.json`.
- **Sync**: Desktop and phone stay synced automatically every few seconds (no manual refresh needed).
- **Add money**: Use `+10`, `+50`, `+100`, or enter a custom amount and click **Add**.
- **Reset**: Click **Reset counter to zero**, then confirm.
- **Switch user**: Click **Switch user** to log in as someone else.
